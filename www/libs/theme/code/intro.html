INTRODUCTION

What is Photroller?

Photroller is the ultimate mobile template using a sidebar, it brings many variations simplicity and extreme speed of loading. The menu can be deployed by tapping the navigation or by simply swiping the page to the right side! We've taken extra care when designing the UI to make sure that your UX is the best it can be!

FEATURES

So what's in this?

Multiple header variations!

The top navigation can either be a huge nice image slider, or you can use solid colors regarding which you chosse, all it takes to change it is adding a simple class!

Solid color headers!

You can also use solid color headers, for secondary pages, not all pages need a big top image slider, so it's much easier to use a solid color.
